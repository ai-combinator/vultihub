Prod releases

Scope: gomes, NeO, Apotheosis, Jibles. Excludes other contributors, bots, the spec repo, and auto-generated dependency files.

  • Jibles — agent transaction tools simplified across the app — replaced a sprawling family of per-chain tools with three unified ones (send / swap / contract call) the agent uses for every chain. Killed a class of "did the swap actually go through?" bugs by making the app wait for chain confirmation before showing a transaction as approved. Started rebuilding how the agent fetches balances so the AI can tell when its numbers are stale and refresh them on demand.
  • gomes + NeO — LUNC support and Cosmos staking — gomes wired the LUNC/USTC bridge to Osmosis end-to-end so users can move funds without leaving the app, plus polish on the receiving-chain side. NeO built Cosmos staking (delegate, undelegate, redelegate, claim rewards) plus a scheduler so users can set up recurring stakes. gomes also fixed a long-standing Ethereum bug where transactions were being silently dropped about 25 blocks in.
  • NeO — massive bug-clearance week — about 25 reliability and design issues fixed across the agent: text in chat now copyable, only the active "thinking" step animates instead of all of them, balances pre-fetch when chat opens, swap confirmation card matches Figma, vault-aware welcome chips, and agent runs no longer cancel when the user backgrounds the app. 59 merged PRs and 43k lines, the most volume on the team.
  • NeO — Security pillar work (proposed mission) — wired in Blockaid (a transaction safety scanner used by Wallet Guard etc.) so risky transactions get flagged before signing. Locked down our analytics so identifying user data can't leak. Forces a fresh biometric check on every signature instead of one-and-done. Blockaid is now in canary rollout.
  • Apotheosis — Vultiverse Airdrop sign-up is live, iOS resubmitted — the system now records who's signed up, separates legacy Station migrants from fresh users, and exposes a public counter for marketing. Quest-tracking landed too. iOS 5.0.0 resubmitted to Apple on Apr 29 with all three review fixes (display name, vault-setup screen, reviewer demo access).
  • Jibles — reliable execution and history storage — making sure the agent's tools actually run and the outcomes get recorded properly. Rewrote how the app reads transactions sent by the server so each chain has its own clean handler (fixes a Solana swap that was breaking on-device). Started moving the record of what the agent has executed off the device and onto the server, so the agent has a single source of truth for action history that survives an app restart.
  • gomes — drove other people's PRs to merge + dogfood polish — took the Cosmos swap signing work over the line (originally NeO's), hardened a Solana fee fix from an external contributor, and shipped 7 small UX fixes from his own dogfood: auth bar back-button keeps you in chat, fee row always shows, sessions panel handles signed-out states cleanly, scheduling screen no longer kicks you out on permission denial.
